# CryptoTrader — Автоматизований торговий бот для ByBit

Проєкт CryptoTrader призначений для автоматичного виконання торгових угод на біржі ByBit (USDT Perpetual) на основі сигналів із Telegram або зовнішніх джерел через REST API.

## 🚀 Основні можливості
- **Автоматичне відкриття позицій:** 10x плече, 1% USDT маржі від балансу.
- **Каскадні тейк-профіти:** Автоматичне виставлення до 4 рівнів TP.
- **Динамічний Stop-Loss:** Переведення в безубиток після TP2, трекінг попередніх рівнів після TP3+.
- **WebSocket моніторинг:** Відстеження виконання ордерів у реальному часі.
- **Відновлення стану:** Автоматична реконциляція з біржею після перезапуску.
- **Сповіщення:** Повний звіт про хід угоди в Telegram.

## ⚙️ Налаштування

Конфігурація знаходиться у файлі `src/CryptoTrader.Main/appsettings.json` або задається через змінні середовища.

### 1. ByBit API
Отримайте ключі в особистому кабінеті ByBit.
```json
"ByBit": {
  "apiKey": "Ваш_Live_Key",
  "apiSecret": "Ваш_Live_Secret",
  "apiKeyDemo": "Ваш_Demo_Key",
  "apiSecretDemo": "Ваш_Demo_Secret",
  "useDemo": "True" // Встановіть False для реальної торгівлі
}
```

### 2. Telegram Бот
Створіть бота через [@BotFather](https://t.me/botfather).
```json
"Telegram": {
  "botKey": "Токен_вашого_бота",
  "privateChatId": "Ваш_Telegram_ID" // Тільки з цього ID бот прийматиме сигнали
}
```

### 3. База даних (PostgreSQL)
```json
"ConnectionStrings": {
  "DefaultConnection": "Host=localhost;Port=5454;Database=cryptotrader_db;Username=trader_admin;Password=YourPassword!"
}
```

## 🛠 Запуск

### Через Docker Compose (рекомендовано)
```bash
docker-compose up -d
```
Це запустить базу даних PostgreSQL та сам додаток. База даних буде доступна на порту `5454`.

### Локальний запуск (.NET 10 SDK)
1. Встановіть залежності: `dotnet restore`
2. Запустіть проєкт: `dotnet run --project src/CryptoTrader.Main/CryptoTrader.Main.csproj`

## 📖 Як працювати з додатком

### Відправка сигналів через Telegram
Просто перешліть або напишіть повідомлення боту у форматі:
```
ZECUSDT SHORT
Вход: 525.98
SL: 530.88
TP1: 522.53
TP2: 519.12
TP3: 517.82
TP4: 517.31
```
*Примітка: Бот підтримує кілька форматів парсингу, включаючи емодзі-варіанти.*

### Відправка сигналів через REST API
Надішліть `POST` запит на `http://localhost:8888/api/v1/signals`:
```json
{
  "text": "BTCUSDT LONG Вход: 65000 SL: 64000 TP1: 66000 TP2: 67000"
}
```

### Моніторинг
- Бот надішле повідомлення `✅ Ордер відкрито`, коли лімітний ордер буде розміщено.
- Бот надішле `📈 Позиція відкрита`, коли ордер виконається (Filled).
- При досягненні TP2 ви отримаєте сповіщення про переміщення SL в точку входу.
- Всі дії логуються у папку `logs/`.

## ⚠️ Важливо
- Бот працює тільки в режимі **One-Way Mode** на ByBit.
- Плече фіксоване — **10x**.
- Ризик на угоду — **1% від капіталу**.
- Для безпеки бот ігнорує повідомлення від усіх користувачів, крім того, чий ID вказано в `privateChatId`.
